## Overview

The Expense Tracker is a responsive React application developed as part of the Syntecxhub Web Development Internship Program.

It helps users manage their daily expenses, monitor spending habits, and visualize financial data through interactive charts. The application provides a simple and clean interface for adding, editing, deleting, and filtering expenses while storing data locally in the browser.

---

## Features

### Expense Management

* Add new expenses
* Edit existing expenses
* Delete expenses
* Categorize expenses
* Search expenses by title

### Dashboard Analytics

* Total Expenses
* Total Transactions
* Highest Expense
* Current Month Spending

### Data Visualization

* Category-wise Pie Chart
* Monthly Expense Bar Chart

### React Hooks Used

* useState
* useEffect
* useRef
* useMemo
* useCallback

### Additional Features

* Local Storage Persistence
* Responsive Design
* Clean User Interface
* Real-Time Updates

## Installation

Clone the repository:

```bash
git clone https://github.com/Kavita2505-eng/Syntecxhub_React_Expense_Tracker.git
```

Navigate to project directory:

```bash
cd Syntecxhub_React_Expense_Tracker
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
